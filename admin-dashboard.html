MANAGEMENT MAP

관리 기능

기능 정의서와 API 명세를 기준으로 구성했습니다.

01

사용자·포인트

전체 사용자 상태와 인가 권한, 고객별 포인트 적립·차감 현황을 관리합니다.

GET /api/v1/admin/users GET /api/v1/admin/users/points PATCH /api/v1/admin/users/{userId}/points
02

상품 운영

전체 상품을 조회하고 불법·위험·신고 상품의 노출을 중지하거나 삭제합니다.

GET /api/v1/admin/products DELETE /api/v1/admin/products/{productId}
03

입점 승인·상태

입점 신청을 승인 또는 반려하고, 기존 입점 업체의 운영 상태를 변경합니다.

GET /api/v1/admin/sellers/applications POST /api/v1/admin/sellers/applications/{appId}/status PUT /api/v1/admin/sellers/{sellerId}/status
04

쿠폰 인가

판매자의 쿠폰 발행 요청을 조회하고 최종 승인 및 인가 처리를 진행합니다.

GET /api/v1/admin/coupons/requests PUT /api/v1/admin/coupons/requests/{requestId}
05

Q&A 모니터링

플랫폼 전체에서 발생한 고객 문의 내역을 조회하고 응대 흐름을 확인합니다.

GET /api/v1/admin/qna
06

플랫폼 정산

전체 거래액, 수수료 수익, 업체별 정산 주기를 확인하고 관리합니다.

GET /api/v1/admin/settlements
